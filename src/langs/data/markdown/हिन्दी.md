# notes.on_mobile_msg

`URL` को `'view-source:'` से उपसर्गित करें। उदाहरण के लिए :- `'https://www.google.com'` -> `'view-source:https://www.google.com'`

# notes.on_computer_msg

मूषिका पर दक्षिण तोदन करें और `'View page source'` को दबाएं।
