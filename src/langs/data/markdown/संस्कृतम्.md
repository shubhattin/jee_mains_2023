# notes.on_mobile_msg

`URLइत्यिदं` `'view-source:'`इत्येन उपसर्गयतु। उदाहराणार्थम् :- `'https://www.google.com'` -> `'view-source:https://www.google.com'`

# notes.on_computer_msg

मूषिकायां दक्षिणतोदनं कुरु `'View page source'`इत्यिदञ्च तुदतु।
