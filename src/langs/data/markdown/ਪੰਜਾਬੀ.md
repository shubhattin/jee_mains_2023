# notes.on_mobile_msg

ਨਾਲ `'view-source:'`ਅਗੇਤਰ `URL` ਕਰੋ। ਉਦਾਹਰਨ ਲਈ :- `'https://www.google.com'` - >`'view-source:https://www.google.com'`

# notes.on_computer_msg

ਮਾਊਸ 'ਤੇ ਰਾਈਟ ਕਲਿੱਕ ਕਰੋ ਅਤੇ ਫਿਰ 'ਤੇ `'View page source'`ਕਲਿੱਕ ਕਰੋ।

# notes.other_notes.0
